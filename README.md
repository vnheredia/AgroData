# 🌱 AGRODATA

Sistema web inteligente para monitoreo climático y predicción de riesgo de heladas en Malchinguí.

## 🚀 Funcionalidades
- Dashboard en tiempo real (temperatura, humedad, viento)
- Semáforo de riesgo (bajo, medio, alto)
- Predicción con Machine Learning
- Historial con filtros por fechas y gráficas
- Mapa de la zona de Malchinguí
- Chatbot con IA (Gemini)

## 🛠️ Tecnologías
- Python (Flask)
- PostgreSQL + PostGIS
- HTML, CSS, JavaScript
- Machine Learning (scikit-learn)
- API Gemini (Google)
- Docker (opcional para despliegue)
